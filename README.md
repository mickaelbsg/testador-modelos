# Testador de Modelos

Interface web para testar modelos LLM, medir latencia, registrar erros e recomendar o melhor modelo disponivel no momento.

## Stack

- Backend: Node.js + Express
- Frontend: React + Vite
- Persistencia: JSON local

## Rodar

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
